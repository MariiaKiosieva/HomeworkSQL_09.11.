z1) Обработка видео 
 а) Update - Пользователь редактирует описание, хештеги, отметки!
 б) Read - Пользователь может может смотреть текущие настройки
 
2) Загрузка видео
 a) Update - Человек может выбирать качество видео при загрузке 
 б) Read - Человек может читать информацию о загруженных видео 
 
3) Коментарии под видео 
 а) Update - Можно коментировать видео 
 б) Read - Можно просто смотреть коментарии 
 
4) Управление каналом
 а) Update - Владелец может редактировать информацию про видео 
 б) Read - может читать информацию про видео 
 
5) Уведомления 
 а) Update - Пользователь может отключать и включать уведомления о других каналах 
 б) Read - Просмотр Уведомлений 
 
6) Поиск видео
 а) Read - можно искать видео по определенным словам тегам и хештегам 
 
7) Лайки под видео 
 а) Update - Можно ставить и убирать лайки под любым видео 
 б) Read - Видеть кто поставил лайк
 
8) подписка на канал 
 а) Update - Пользователь может подписаться на любой канал 
 б) Read - так же модет видеть пользователей которые также подписаны
